# Lotto Number Recommender

A simple web application that recommends 6 unique lotto numbers (1-45) and displays them using a custom Web Component.

## Features

- Generates 6 unique random numbers between 1 and 45.
- Displays numbers in ascending order.
- Visualizes numbers as colored balls using a custom `<lotto-ball>` Web Component.
- Range-based coloring for lotto balls:
  - 1-10: Yellow
  - 11-20: Blue
  - 21-30: Red
  - 31-40: Gray
  - 41-45: Green

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+ Modules, Web Components)

## How to Run

Simply open `index.html` in a modern web browser that supports ES6 modules and Web Components.
